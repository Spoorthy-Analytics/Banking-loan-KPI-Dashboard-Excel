# Banking Loan KPI Dashboard | Excel

## Project Overview

This Excel-based Banking Loan KPI Dashboard provides insights into loan performance, customer activity, repayment behavior, and branch-level loan distribution. The dashboard helps financial institutions monitor lending operations and identify areas for risk management and business growth.

## Key Business Questions Answered

2. Which branches have the highest loan distribution?
